# 香港六合彩預測系統

這個資料夾是手機獨立雲端發布資料。手機網址：

```text
https://pingshen670924-dotcom.github.io/-/
```

同步規則：

- 電腦端一鍵啟動完成後，會把同一份手機頁、完整戰報、最新預測、歷史資料同步到此資料夾並發布。
- GitHub Pages 的 `index.html` 只是雲端必要入口，正式手機頁檔名為 `香港六合彩預測系統_手機首頁.html`。
- GitHub Actions 在本機同步 push 時只部署，不重新運算，確保手機版與電腦版一致。
- 排程或手動工作流程執行時，雲端才會自行抓取開獎並重新運算。

正式顯示名稱只使用 `香港六合彩預測系統`。
